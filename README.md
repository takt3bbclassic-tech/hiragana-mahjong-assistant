# Hiragana Mahjong Solo

A solo practice web app for a hiragana tile word-building game.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel

Recommended Vercel settings:

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

## Notes

This repository intentionally uses a generic project name and avoids official branding in the app title.
